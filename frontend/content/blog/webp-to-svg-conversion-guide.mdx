## WebP란 무엇이고, 왜 SVG로 변환해야 할까?

WebP는 Google이 개발한 차세대 이미지 포맷으로, PNG나 JPG보다 파일 크기가 작으면서도 화질을 유지하는 것이 특징입니다. 웹 성능 최적화에 탁월하지만 **확대하면 픽셀이 깨지는 래스터 이미지**라는 한계가 있습니다.

반면 SVG(Scalable Vector Graphics)는 수학적 경로로 이미지를 정의하기 때문에 **어떤 크기로 확대해도 선명함을 유지**합니다. 로고, 아이콘, 일러스트 등에 WebP 대신 SVG를 사용하면 디자인 품질이 크게 향상됩니다.

---

## WebP vs SVG 비교

| 항목 | WebP | SVG |
|------|------|-----|
| 파일 형식 | 래스터(픽셀) | 벡터(경로) |
| 확대 시 품질 | 픽셀 깨짐 | 무한 선명 |
| 파일 크기 | 작음 | 복잡도에 따라 다름 |
| 주요 용도 | 웹 사진, 배너 | 로고, 아이콘, UI 요소 |
| CSS/JS 조작 | 불가 | 가능 |
| 브라우저 지원 | 모든 최신 브라우저 | 모든 최신 브라우저 |

## WebP를 SVG로 변환하는 단계별 방법

### 1단계: 변환 툴 접속

[imgtosvg](https://img-to-svg-six.vercel.app)에 접속합니다. 회원가입이나 프로그램 설치 없이 바로 사용할 수 있습니다.

### 2단계: WebP 파일 업로드

페이지 중앙의 업로드 영역에 WebP 파일을 드래그 앤 드롭하거나, 클릭해서 파일을 선택합니다.

### 3단계: 변환 옵션 선택 (선택사항)

배경 제거가 필요한 경우 **배경 제거 옵션**을 활성화할 수 있습니다. 로고나 아이콘 변환 시 특히 유용합니다.

### 4단계: SVG 다운로드

변환이 완료되면 SVG 파일을 다운로드합니다. 변환 속도는 이미지 복잡도에 따라 수 초 내에 완료됩니다.

## 변환 시 주의사항

- **단순한 이미지일수록 결과물이 좋습니다.** 로고, 아이콘, 일러스트 등 색상이 명확히 구분된 이미지에서 최상의 결과를 얻을 수 있습니다.
- **사진처럼 복잡한 이미지**는 SVG 파일 크기가 커질 수 있으며, 결과물의 정밀도가 낮아질 수 있습니다.
- 변환 후 **피그마, 일러스트레이터 등 벡터 편집 툴**에서 추가 수정이 가능합니다.

## 자주 묻는 질문 (FAQ)

### Q1. WebP를 SVG로 변환해도 화질이 유지되나요?

네, SVG는 벡터 형식이기 때문에 변환 후에는 어떤 크기로 확대해도 선명하게 유지됩니다. 다만 원본 WebP의 복잡도에 따라 경로 세부 표현에 차이가 있을 수 있습니다.

### Q2. 무료로 변환할 수 있나요?

imgtosvg는 완전 무료로 제공됩니다. 가입 없이 즉시 사용할 수 있으며, 변환 횟수에 제한이 없습니다.

### Q3. 변환한 SVG 파일을 피그마에서 사용할 수 있나요?

네, 변환된 SVG 파일은 피그마, 일러스트레이터, 잉크스케이프 등 모든 벡터 편집 툴에서 바로 열 수 있습니다.

### Q4. 배경이 있는 WebP도 SVG로 변환할 수 있나요?

가능합니다. 배경 제거 옵션을 활성화하면 배경 없이 주요 오브젝트만 SVG로 추출할 수 있어, 로고나 스티커 디자인에 매우 유용합니다.

### Q5. WebP 외에 어떤 형식을 지원하나요?

imgtosvg는 WebP 외에도 PNG, JPG 파일의 SVG 변환을 지원합니다.
