## JPG와 SVG의 차이점

JPG는 사진에 특화된 압축 포맷이고, SVG는 수학 공식으로 그린 벡터 이미지입니다.

| 구분 | JPG | SVG |
|------|-----|-----|
| 방식 | 픽셀(점) 집합 | 수학 공식(벡터) |
| 확대 시 | 깨짐 | 선명함 유지 |
| 투명도 | 미지원 | 지원 |
| 편집 | 포토샵 필요 | 피그마에서 바로 편집 |
| 적합한 이미지 | 사진 | 로고, 아이콘, 일러스트 |

**핵심:** 로고나 아이콘을 JPG로 받았다면 SVG로 변환해야 피그마에서 편집하고 어떤 크기로 써도 선명하게 출력할 수 있습니다.

## JPG를 SVG로 변환해야 하는 이유

이런 상황이라면 JPG → SVG 변환이 필요합니다:

- **피그마 작업 시** — 클라이언트에게 받은 JPG 로고를 수정해야 할 때
- **웹 개발** — `<img>` 태그 대신 인라인 SVG로 CSS 애니메이션 적용 시
- **인쇄물/현수막** — 대형 출력에서 품질 저하 없이 사용 시
- **PPT/키노트** — 배경 투명한 로고 이미지 삽입 시

## 무료로 JPG SVG 변환하는 방법

### 준비물
- 변환할 JPG 파일 (최대 10MB)
- 인터넷 브라우저 (별도 설치 불필요)

### 단계별 방법

**① imgtosvg 접속**

[imgtosvg](/) 에 접속합니다. 로그인이나 회원가입이 **전혀 필요 없습니다.**

**② JPG 파일 업로드**

드래그앤드롭 영역에 JPG 파일을 끌어다 놓거나 "파일 선택하기" 버튼을 클릭합니다.

지원 형식: JPG, PNG, WebP (최대 10MB)

**③ 옵션 선택**

- **배경 제거 체크** — JPG는 배경이 흰색인 경우가 많습니다. 체크하면 AI가 배경을 자동으로 제거해 투명 SVG를 만들어줍니다.
- **색상 정밀도** — 로고/아이콘은 낮게(3~4), 복잡한 이미지는 높게(6~8)

**④ SVG로 변환 클릭**

"SVG로 변환" 버튼을 누르면 수초 내에 변환이 완료됩니다.

**⑤ 다운로드**

"SVG 다운로드" 버튼으로 저장합니다. 업로드 파일은 변환 완료 즉시 자동 삭제됩니다.

## JPG 이미지 유형별 최적 설정

**단순 로고 / 아이콘 (선이 명확한 이미지)**
- 색상 정밀도: 3~4
- 잡음 제거: 높게 (10 이상)
- 배경 제거: 흰 배경이면 체크

**복잡한 일러스트 / 그래픽**
- 색상 정밀도: 6~8
- 잡음 제거: 낮게 (1~3)
- 결과: 원본 색상에 가까운 SVG

**사진 이미지 (JPG 특유의 경우)**
> 사진은 픽셀 수가 매우 많아 SVG 변환 결과가 복잡해질 수 있습니다. 로고나 일러스트에 가장 좋은 결과를 얻을 수 있습니다.

## 변환한 SVG 피그마에서 사용하는 방법

1. 피그마에서 새 파일 열기
2. 변환된 `.svg` 파일을 캔버스에 **드래그앤드롭**
3. 불러온 SVG 더블클릭 → 개별 path 선택 가능
4. 색상, 크기, 모양 자유롭게 수정

SVG를 제대로 변환했다면 더블클릭 시 내부 요소가 각각 선택됩니다. 이게 JPG와의 결정적 차이입니다.

## 자주 묻는 질문

**Q. JPG SVG 변환은 무료인가요?**

네, 완전 무료이며 회원가입도 필요 없습니다.

**Q. 변환 품질이 낮게 나왔어요. 어떻게 하나요?**

색상 정밀도를 높이고, 잡음 제거 수치를 낮춰보세요. 원본 이미지가 로고/아이콘처럼 단순할수록 결과가 좋습니다.

**Q. 업로드한 JPG 파일은 서버에 저장되나요?**

아니요. 변환 완료 즉시 자동 삭제됩니다.

**Q. PNG와 JPG 중 어떤 걸 변환하는 게 더 좋은가요?**

가능하면 PNG로 변환하는 것이 더 좋습니다. PNG는 무손실 압축이라 선이 더 선명하게 변환됩니다. JPG는 압축 과정에서 노이즈가 생겨 SVG 경로가 복잡해질 수 있습니다.
