## SVG 파일이란?

**SVG(Scalable Vector Graphics)** 는 XML 기반의 벡터 이미지 형식입니다. 픽셀 대신 수학적 좌표와 경로로 이미지를 표현하기 때문에, 아무리 크게 확대해도 이미지가 흐릿해지지 않습니다. 웹 아이콘, 로고, 일러스트 등에 널리 활용됩니다.

## SVG vs PNG vs JPG 비교

| 항목 | SVG | PNG | JPG |
|------|-----|-----|-----|
| 파일 형식 | 벡터(XML) | 비트맵(픽셀) | 비트맵(픽셀) |
| 확대 시 품질 | 손실 없음 | 깨짐 발생 | 깨짐 발생 |
| 투명 배경 | 지원 | 지원 | 미지원 |
| 파일 크기 | 보통 작음 | 보통 큼 | 가장 작음 |
| 애니메이션 | CSS/JS로 가능 | 불가 | 불가 |
| 웹 사용 | 매우 적합 | 적합 | 사진에 적합 |

## SVG 파일의 주요 장점

### 1. 무한 확대 가능 (해상도 독립)
SVG는 벡터 기반이라 모바일 화면부터 대형 현수막까지 단 하나의 파일로 대응할 수 있습니다. 레티나 디스플레이에서도 선명하게 표시됩니다.

### 2. 파일 크기 절감
간단한 아이콘이나 로고는 PNG보다 SVG가 훨씬 파일 크기가 작아 웹 페이지 로딩 속도를 높입니다.

### 3. CSS·JavaScript로 제어 가능
SVG는 코드이기 때문에 색상·크기·애니메이션을 CSS나 JS로 동적으로 바꿀 수 있습니다. 다크 모드 아이콘 전환에도 매우 유용합니다.

### 4. 검색엔진 친화적
SVG 내부 텍스트는 검색엔진이 읽을 수 있어 SEO에 유리합니다.

## SVG 파일은 어디에 쓰이나요?

- **웹 아이콘 및 UI 요소** — 버튼, 메뉴 아이콘, 로딩 스피너
- **브랜드 로고** — 인쇄·디지털 모두 단일 파일 사용
- **인포그래픽·차트** — D3.js 등 데이터 시각화 라이브러리
- **애니메이션 배너** — CSS 애니메이션과 결합
- **피그마·Adobe XD 내보내기** — 디자인 툴 기본 출력 형식

## FAQ

### Q. SVG 파일은 어떤 프로그램으로 열 수 있나요?
A. 모든 최신 브라우저(Chrome, Firefox, Safari, Edge)에서 바로 열 수 있습니다. 편집은 피그마, Adobe Illustrator, Inkscape(무료) 등으로 할 수 있습니다.

### Q. 사진(풍경, 인물 사진)도 SVG로 변환할 수 있나요?
A. 가능하지만 권장하지 않습니다. 사진처럼 복잡한 픽셀 정보는 SVG로 변환하면 파일이 오히려 커지고 품질도 떨어집니다. SVG는 단순한 형태의 로고·아이콘·일러스트에 가장 적합합니다.

### Q. SVG 파일을 PNG로 다시 바꿀 수 있나요?
A. 네. 브라우저에서 SVG를 열고 우클릭 → '다른 이름으로 저장'을 사용하거나, 온라인 변환 도구를 이용하면 됩니다.

### Q. SVG는 모든 브라우저에서 지원되나요?
A. Internet Explorer 8 이하를 제외한 모든 현대 브라우저에서 완벽히 지원됩니다. 현재 IE는 거의 사용되지 않으므로 실제 사용 환경에서는 문제가 없습니다.

### Q. SVG 파일 내부 코드를 직접 편집할 수 있나요?
A. 네. SVG는 텍스트(XML)이기 때문에 메모장이나 VS Code로 열어 색상 코드·크기·경로를 직접 수정할 수 있습니다.
