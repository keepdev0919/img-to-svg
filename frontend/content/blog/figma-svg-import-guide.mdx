## 피그마에서 SVG 파일을 불러오는 방법

피그마(Figma)는 웹 기반 디자인 툴로, SVG 파일을 네이티브로 지원합니다. SVG를 불러오면 개별 패스(path)와 레이어를 자유롭게 편집할 수 있어 디자이너들에게 매우 유용합니다.

### SVG 파일 가져오기 기본 단계

1. 피그마 캔버스를 열고 **File → Place image…** 를 선택합니다.
2. 탐색기에서 `.svg` 파일을 선택한 뒤 **열기**를 클릭합니다.
3. 캔버스 위를 클릭하면 SVG가 프레임으로 삽입됩니다.
4. 삽입된 SVG를 마우스 오른쪽 버튼으로 클릭 → **Ungroup** 또는 **Flatten**을 선택하면 개별 벡터 레이어로 분리됩니다.

> **팁:** 드래그 앤 드롭으로도 SVG를 바로 캔버스에 끌어다 놓을 수 있습니다.

## SVG vs 래스터 이미지 — 피그마 편집 비교

| 항목 | SVG (벡터) | PNG / JPG (래스터) |
|------|-----------|------------------|
| 해상도 손실 | 없음 (무한 확대 가능) | 확대 시 픽셀화 발생 |
| 레이어 분리 편집 | 가능 | 불가능 |
| 파일 크기 | 복잡도에 따라 작음 | 해상도에 비례해 큼 |
| 색상·형태 수정 | 피그마 내에서 직접 수정 | 재출력 필요 |
| 애니메이션 적용 | CSS·JS로 활용 가능 | 제한적 |

## PNG·JPG를 SVG로 변환한 뒤 피그마에서 활용하기

로고나 아이콘이 래스터 포맷(PNG, JPG)으로만 있다면, 먼저 SVG로 변환해야 피그마에서 벡터 편집이 가능합니다.

### 변환 절차

1. **[imgtosvg](https://img-to-svg-six.vercel.app)** 에 접속합니다.
2. PNG 또는 JPG 이미지를 업로드합니다.
3. 변환 옵션을 선택(색상 수, 경로 세밀도 등)하고 **변환** 버튼을 클릭합니다.
4. 생성된 `.svg` 파일을 다운로드합니다.
5. 다운로드한 SVG를 피그마에 드래그 앤 드롭하면 바로 벡터 레이어로 사용 가능합니다.

## 자주 묻는 질문 (FAQ)

### Q1. 피그마에 SVG를 불러왔는데 편집이 안 돼요.
**A.** SVG가 단일 이미지로 임포트된 경우입니다. 레이어 패널에서 해당 요소를 선택한 뒤 **오른쪽 클릭 → Ungroup**을 반복하거나, **Double click**으로 그룹을 해제해 보세요.

### Q2. SVG 파일 크기가 너무 커서 피그마가 느려요.
**A.** 경로(path)가 지나치게 복잡한 SVG는 성능 저하를 일으킵니다. imgtosvg에서 변환할 때 **경로 단순화 옵션**을 높이거나, 피그마의 **Flatten Selection** 기능으로 패스를 병합하면 개선됩니다.

### Q3. 피그마에서 SVG를 내보내는 방법은?
**A.** 내보낼 레이어 또는 프레임을 선택 → 오른쪽 패널 하단 **Export** 섹션에서 형식을 **SVG**로 선택 → **Export** 버튼을 클릭합니다.

### Q4. 무료 온라인 툴로 PNG를 SVG로 변환할 수 있나요?
**A.** 네, **[imgtosvg](https://img-to-svg-six.vercel.app)** 는 완전 무료로 PNG, JPG, WebP를 SVG로 변환해 줍니다. 별도 설치 없이 브라우저에서 바로 사용할 수 있습니다.

### Q5. 피그마 무료 플랜에서도 SVG를 불러올 수 있나요?
**A.** 네, SVG 임포트·익스포트 기능은 무료 플랜(Starter)에서도 제한 없이 사용 가능합니다.
