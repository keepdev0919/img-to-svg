## PNG와 SVG의 차이점

PNG와 SVG는 둘 다 이미지 파일이지만 **저장 방식이 완전히 다릅니다.**

| 구분 | PNG | SVG |
|------|-----|-----|
| 방식 | 픽셀(점) 집합 | 수학 공식(벡터) |
| 확대 시 | 깨짐 | 선명함 유지 |
| 파일 크기 | 큼 | 작음 (단순 이미지) |
| 편집 | 포토샵 필요 | 피그마에서 바로 편집 |
| 웹 사용 | O | O (더 가볍게) |

**핵심:** 로고, 아이콘, 일러스트는 SVG로 만들면 어떤 크기로 써도 선명하고 피그마에서 자유롭게 수정할 수 있습니다.

## PNG를 SVG로 변환해야 하는 이유

이런 상황이라면 PNG → SVG 변환이 필요합니다:

- **피그마 작업 시** — 클라이언트에게 받은 로고 PNG를 편집해야 할 때
- **PPT/키노트 발표** — 배경 투명한 로고를 고해상도로 삽입할 때
- **웹 개발** — 레티나 디스플레이에서 깨지지 않는 아이콘이 필요할 때
- **인쇄물 작업** — 명함, 현수막 등 대형 출력 시 품질 유지

## 무료로 PNG SVG 변환하는 방법

### 준비물
- 변환할 PNG 또는 JPG 파일 (최대 10MB)
- 인터넷 브라우저 (설치 불필요)

### 단계별 방법

**① imgtosvg 접속**

[imgtosvg](/) 에 접속합니다. 회원가입이나 로그인이 **전혀 필요 없습니다.**

**② 파일 업로드**

"이미지를 드래그하거나 클릭해서 선택" 영역에 PNG 파일을 끌어다 놓거나 클릭해서 선택합니다.

지원 형식: PNG, JPG, WebP (최대 10MB)

**③ 옵션 설정 (선택사항)**

기본 설정으로도 충분하지만, 더 나은 결과를 위해:
- **색상 정밀도** — 숫자가 높을수록 색상이 풍부하게 표현됩니다
- **배경 제거** — 체크하면 흰 배경을 자동으로 제거합니다

**④ SVG로 변환 클릭**

"SVG로 변환" 버튼을 누르면 수초 내에 변환이 완료됩니다.

**⑤ 다운로드**

"SVG 다운로드" 버튼으로 파일을 저장합니다. **업로드된 파일은 변환 후 즉시 서버에서 삭제**되어 개인정보가 안전하게 보호됩니다.

## 변환 품질을 높이는 설정 팁

PNG 이미지 유형별로 최적 설정이 다릅니다:

**로고/아이콘 (단순한 이미지)**
- 색상 정밀도: 3~4 (낮게)
- 잡음 제거: 높게
- 결과: 깔끔한 선, 작은 파일 크기

**사진/복잡한 이미지**
- 색상 정밀도: 6~8 (높게)
- 잡음 제거: 낮게
- 결과: 원본에 가까운 색상 표현

**배경 제거가 필요한 경우**
- "배경 제거" 옵션 체크
- AI가 자동으로 배경을 인식하고 투명하게 처리합니다

## 피그마에 SVG 불러오는 방법

변환한 SVG를 피그마에서 편집하려면:

1. 피그마 캔버스 열기
2. 다운로드한 `.svg` 파일을 캔버스로 **드래그앤드롭**
3. 불러온 SVG를 더블클릭 → 개별 path 선택 가능
4. 색상, 크기, 모양을 자유롭게 편집

SVG가 제대로 변환됐다면 **더블클릭 시 내부 요소가 개별 선택**됩니다. 이것이 PNG와의 결정적 차이입니다.

## 자주 묻는 질문

**Q. PNG SVG 변환은 무료인가요?**

네, imgtosvg는 완전 무료이며 회원가입도 필요 없습니다.

**Q. JPG 파일도 SVG로 변환할 수 있나요?**

네, PNG뿐만 아니라 JPG, WebP 형식도 지원합니다. 변환 방법은 동일합니다.

**Q. 변환한 SVG 파일이 깨져 보여요. 왜 그런가요?**

원본 이미지가 복잡한 사진이라면 SVG 변환 품질이 낮아질 수 있습니다. SVG는 단순한 로고, 일러스트, 아이콘에 최적화되어 있습니다.

**Q. 업로드한 이미지는 어떻게 처리되나요?**

모든 업로드 파일은 변환 완료 즉시 서버에서 자동 삭제됩니다.

**Q. 피그마 말고 일러스트레이터에서도 열 수 있나요?**

네, SVG는 어도비 일러스트레이터, 잉크스케이프, 스케치 등 모든 벡터 편집 프로그램에서 열 수 있습니다.
